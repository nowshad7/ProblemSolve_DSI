## Vehicle Showroom (Console App)
### Tools:
- Python 3
- Mysql

### Function:
- Add Vehicle
- Remove Vehicle
- Show All
- Expected Visitor

### Classes:
- Vehicle (Parent)
- SportsVehicle (Child)
- HeavyVehicle (Child)

### Database:
- Name: vehicle
- Tables
    - NormalVehicle(id, model, eType, power, tire)
    - SportsVehicle(id, model, eType, power, tire, Turbo)
    - HeavyVehicle(id, model, eType, power, tire, weight)
